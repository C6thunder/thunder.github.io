# 🎨 头像修改说明

## 📁 统一配置

现在所有页面的头像都通过 `config.js` 统一管理，只需要修改一个文件即可更新全站头像！

## 🔧 修改头像的方法

### 方法1：直接修改 config.js（推荐）

编辑 `config.js` 文件，找到这一行：
```javascript
src: 'img/avatar.jpg?v=1762753846',
```

替换为你的头像路径，例如：
```javascript
src: 'img/my-avatar.jpg',  // 本地图片
// 或者
src: 'https://example.com/avatar.jpg',  // 在线图片
```

### 方法2：通过浏览器控制台（临时）

打开浏览器开发者工具（按F12），在Console中输入：
```javascript
updateAvatar('img/new-avatar.jpg?v=12345678')
```

**注意**：这个方法只对当前页面有效，刷新后会恢复。

### 方法3：通过 localStorage（本地存储）

在控制台输入：
```javascript
localStorage.setItem('customAvatar', 'img/new-avatar.jpg')
location.reload()
```

这会将自定义头像保存在浏览器中。

## 📂 放置头像文件

将头像文件放在以下位置：
```
/home/thunder/work/Web/thunder.github.io/
└── img/
    └── avatar.jpg  ← 放在这里
```

## 🎯 支持的头像格式

- JPG / JPEG
- PNG
- GIF
- WebP
- 在线图片URL

## ✅ 修改后的效果

修改 `config.js` 后，刷新页面，所有页面的头像都会自动更新：
- 首页左下角
- 笔记列表页左下角
- 写笔记页右上角
- 个人页左下角和中央大头像
- 笔记详情页左下角

## 🆘 如果头像不更新

1. 按 `Ctrl+F5` 强制刷新
2. 清除浏览器缓存
3. 检查头像文件路径是否正确
4. 确认 `config.js` 已正确加载（开发者工具Console中输入 `window.APP_CONFIG` 查看）

## 📝 其他可配置的信息

在 `config.js` 中还可以修改：
- 网站标题
- 作者姓名
- 邮箱地址
- GitHub链接
- 默认主题
