JavaScript 基础学习笔记

变量声明

JavaScript中有三种声明变量的方式：

1. var

  • 函数作用域
  • 可以重复声明
  • 有变量提升

2. let

  • 块级作用域
  • 不能重复声明
  • 暂存死区

3. const

  • 块级作用域
  • 必须初始化
  • 不能重新赋值

函数

函数是JavaScript的基础构件。

函数声明

function greet(name) {


return Hello, ${name}!;


}

箭头函数

const greet = (name) => Hello, ${name}!;
